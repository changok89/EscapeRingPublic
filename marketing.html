탈출링 (EscapeRing)

불편한 순간, 자연스럽게 빠져나오는 가짜전화 시뮬레이터. 원하는 시점에 리얼한 착신 화면과 통화 연출을 시작하세요.

핵심 기능

  • 즉시 호출 / n분 후 / 특정 시간 예약 호출
  • 착신자 이름·번호·테마를 자유롭게 설정
  • 통화 스크립트 TTS(한국어/영어/일본어) 지원
  • 최대 5문장 직접 녹음 + 최대 7문장 텍스트 구성
  • 자동 재생 / 탭 진행 방식 선택
  • 백그라운드 타이머와 알림 기반 실행

사용 흐름

시나리오 선택 또는 커스텀 작성 → 타이머 설정 → 착신 화면 시작 → 통화 연출 → 종료

본 앱은 실제 통화 서비스가 아닌 시뮬레이션 앱입니다.

이런 상황에서 유용합니다

  • 회식, 모임, 미팅에서 자연스러운 자리 이탈이 필요할 때
  • 소개팅, 약속 등에서 부드러운 종료 타이밍이 필요할 때
  • 콘텐츠 촬영/연습용 통화 연출이 필요할 때
